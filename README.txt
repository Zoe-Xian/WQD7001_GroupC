Step 1: Install the Python Environment
Make sure Python is installed on your computer (recommended version >= 3.8).

Step 2: Install dependent libraries
Run in the project root directory:
pip install -r requirements.txt

Step 3: Run the Flask Application
In the project root directory, run the following command to start the Flask application:
python app.py

Step 4: Access the local service
Open http://127.0.0.1:5000 in your browser to access the application.
